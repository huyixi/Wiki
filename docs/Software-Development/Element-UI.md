# ElementUI

Element-ui最终的形式是一个外部的新vue组件，可添加集成到Vue主组件中，element-ui这个组件内部也包含着很多小的Vue组件，例如Element经常使用的Button等组件。Element既能全局引入，也能单独引入需要的组件，其本质就是因为，整个element-UI是一个大的vue组件，每个小的功能组件也是一个单独的vue组件。
